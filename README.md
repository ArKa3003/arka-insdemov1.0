# ARKA-INS Demo

Prior Authorization Intelligence Platform for Radiology Benefit Managers

## Features

- Pre-denial appeal risk prediction
- CMS-0057-F compliance tracking
- Gold card eligibility management
- Human-in-the-loop audit trails
- Provider performance analytics

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel --prod
```

## Demo Scenarios

1. **Standard Flow**: Well-documented approval
2. **High-Risk Case**: Appeal risk prediction in action
3. **Gold Card Provider**: Instant auto-approval
